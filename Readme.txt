				Guide for Installing this Pack:

1. Make sure you have installed Microsoft SQL Server 2000 SP4 or newer
2. Create new DataBase named MuOnline. Restore from folder. We suggest you to restore MD5- version.
3. Press Windows+R there enter "odbcad32.exe" without quotes. In User DSN create ODBC names. 
(you can see them in odbc.txt) 
4. Change 10.10.10.10* to your IP in ServerList.dat (or BoR_ServerList.dat), MapServerInfo.dat 
5. Change PASS* of you sa user in this files: CashShopOption.ini, svconfig.ini (Event), svconfig.ini 
(Ranking), exdb.ini, GameServer.ini & GameServerCS.ini
6. Your server is ready to use. Check for updates every week.

*Warning: Don't change them somewhere else!!!
**Sometimes antivirus software falsely take the server components like virus files. 
Don't afraid this is just reaction for PE Rebuild or Packers.

				Switch Seasons from 4.6 to 4.5 [ENG]:
1. Remove Data folder
2. Rename DataS4.5 to Data
3. Change Client Serial and Version in GreatDevelop\Common.ini
4. Switch off MossGambler and DuelManager
5. Enjoy bugless Season 4.5 :)
